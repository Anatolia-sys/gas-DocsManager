# 議事録管理アプリ 追加要望

- テンプレートに関してGoogleドキュメントをテンプレートとして登録する仕様に変更する
- フィルター機能と検索機能の導入
- WebAppで初回起動時にGoogleスプレッドシートを指定のフォルダに追加する仕組み
- 案件名も管理し、紐づいたフォルダに自動で議事録を生成するようにしたい。案件ごとにどのフォルダに設定するかはユーザーが一律管理できるようにしたい。
- デザインに関して、もう少しモダンな感じのアプリにしたい。Claudeのようなデザインにしてほしい。
- 会議に関して参加しているメンバー情報も記録し、簡単に議事録をgmailかwebhookを取得したgooglechatで配布できる仕組みを追加したい
